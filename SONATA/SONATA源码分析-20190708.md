# SONATA源码分析
## 1. 目录结构
```js
sonata/
  - core/                // 运行时、Query Partition功能
    - lp/                // 整数规划
    - training/          // 历史数据训练
    - runtime.py         // 运行时
    - refinement.py      // 优化
    - partition.py       // 查询任务分割
    - utils.py           // 其他功能封装
  - query_engine/        // 可声名式抽象查询
    - sonata_operators/  // Query操作符(map、filter、join等)封装
    - sonata_queries.py  // PacketStream类封装
    - utils.py           // Query其他功能封装
  - dataplane_driver/    // 数据平面驱动
  - streaming_driver/    // 流处理器驱动
```
